Inventory System

Licenses System

General-purpose licensing built on SrvData keys with dual states for driving school (Temporary vs Final). Includes staff commands to grant, revoke, or return categories, plus a dropdown UX for selective removals. Fully logged and compatible with group-based permissions.

Discord Log System

All-in-one logging resource that standardizes rich embeds across the server (payments, robberies, tuning, admin actions). Single script handles channels, markdown formatting, rate-limit safety, and consistent fields (player name, passport, amounts, items, plate, etc.), making audit trails clear and searchable.

Wall Script

Staff-only overlay that displays floating player names, source IDs, and passports above heads—visible only to staff who enable it. Shows a red [WALL] tag when both staff members have the tool active, with crisp outlines and centered typography for readability.

Armored Truck Heist

Stockade robbery built from scratch: crowbar requirement, server-driven progress, milestone police alerts, and global cooldowns. Integrates a dedicated Robber skill with XP/level-ups, plus dynamic risk/reward scaling and clean, staff-friendly logging.

Skill / XP System

Universal progression framework where players level roles like Robber and Miner. Grants tangible bonuses (reduced action time, payout multipliers), with clear notifications, persistent storage, and easy hooks for new activities and jobs.

About Gabriel

Designer, programmer, and community builder from Portugal

I am Gabriel "Synthix" Silva, a Portuguese gameplay programmer, designer, and content creator. I thrive on crafting unique concepts and taking on challenges across design, coding, and gaming.

I have been scripting in Lua/FiveM for over 5 years with a deep focus on roleplay systems. Before that, I was building mods and scripts from age 11, which taught me how to design core gameplay mechanics and multiplayer systems from the ground up.

Design has always been part of the journey: I started in Photoshop back when the logo was still the feather, sharpening my eye for branding, UI polish, and advertisement that now elevates every game project I touch.

Today I’m transitioning that systems mindset into Unreal Engine, blending Blueprints and C++ toward fully remote roles across indie, startup, and tools teams. Alongside Medusa development I publish a daily Dev Diary sharing tasks, progress, and learnings with the community.

Objective

My goal is to secure a full-time remote gameplay programming role where I can grow and evolve while contributing to live projects — moving beyond part-time work and applying the multiplayer systems expertise I’ve built through years of server development.
